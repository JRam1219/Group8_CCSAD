SharePoint 2010 Branding and User Interface Design

By Rand Drisgill, John Ross, and Jacob J. Stanford

Synopsis:

SharePoint 2010 Branding and User Interface Design by Rand Drisgill, John Ross, and Jacob J. Stanford is a comprehensive guide for those looking to master SharePoint branding and create custom user interfaces for SharePoint 2010. This book focuses on the design aspects of SharePoint, including customizing themes, integrating branding elements, and designing a user-friendly interface. With practical examples and step-by-step instructions, readers will learn to create visually appealing and functional SharePoint sites that enhance user experience and meet business needs.

Overview:

  • Introduction to SharePoint Branding: Covers the basics of SharePoint 2010 branding and user interface design principles.
  • Custom Themes and Master Pages: Guides readers on creating and modifying SharePoint themes and master pages to align with corporate branding.
  • Designing User Interfaces: Focuses on creating an intuitive and visually consistent user interface for SharePoint sites.
  • Advanced Branding Techniques: Explores more advanced branding techniques, including custom page layouts, web parts, and CSS customization.
  • Integration with SharePoint Features: How to seamlessly integrate custom branding with SharePoint’s out-of-the-box features like lists, libraries, and workflows.
  • Testing and Troubleshooting: Techniques for testing and troubleshooting SharePoint branding implementations to ensure compatibility and usability.

This book is an essential resource for SharePoint designers and developers looking to enhance the branding and usability of their SharePoint 2010 sites, whether for internal or external-facing applications.
